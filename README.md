# titanic-kaggle

Solución al desafío [Titanic - Machine Learning from Disaster](https://www.kaggle.com/c/titanic) de Kaggle.

## Estructura del proyecto

```
├── notebooks/       # Jupyter notebooks de exploración y modelado
├── data/            # Datos crudos y procesados
├── submissions/     # Archivos de submisión generados
├── requirements.txt # Dependencias de Python
└── README.md
```

## Instalación

```bash
pip install -r requirements.txt
```
